
#

## 异常类型

| 类型                       | 是否能捕获 | 原因     | 解决方式 |
| -------------------------- | ---------- | -------- | -------- |
| 除0                        | 否         | 浮点异常 | SIGFPE   |
| 空指针                     | 否         | 段错误   | SIGSEGV  |
| 访问类实例没有初始化的变量 | 否         |          |          |
